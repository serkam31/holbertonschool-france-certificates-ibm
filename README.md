# Holberton School France - Certificats IBM

Ce repository contient les certificats obtenus dans le cadre du programme de formation IBM à Holberton School France.

## Structure du Repository

```
holbertonschool-france-certificates-ibm/
├── certificates-trimester-1/    # Certificats du premier trimestre
│   └── certificate-agile       # Certificat Agile Development
└── certificate-agile.pdf       # Certificat Agile (format PDF)
```

## Certificats Disponibles

### Trimestre 1

- **Agile Development** - Méthodologies et pratiques de développement Agile
- **Open Source Software Development** - Développement de logiciels open source

## À Propos

Ce repository fait partie du cursus de formation à Holberton School France, en partenariat avec IBM, visant à former les étudiants aux meilleures pratiques de l'industrie du développement logiciel.
